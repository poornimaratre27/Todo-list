# TaskFlow – Advanced To-Do Dashboard

TaskFlow is a responsive and modern To-Do Dashboard built using HTML, CSS, and JavaScript. It helps users manage daily tasks efficiently with real-time analytics, priority tracking, progress visualization, and customizable settings.

---

## ✨ Features

### 📋 Task Management

* Add new tasks
* Edit existing tasks using popup modal
* Delete tasks
* Mark tasks as completed or pending
* Real-time task updates
* Enter key support for quick task addition

### 🎯 Task Organization

* High / Medium / Low priority levels
* Due date selection
* Overdue task detection
* Search tasks instantly
* Filter:

  * All Tasks
  * Completed Tasks
  * Pending Tasks

### 📊 Analytics Dashboard

* Overall task progress graph
* Completed vs Pending visualization
* Priority-wise analytics
* Interactive priority cards
* Progress percentage tracking

### ⚙️ Settings

* Dark/Light theme toggle
* Export tasks as JSON
* Import tasks from JSON
* Clear all tasks
* Reset application

### 💾 Storage

* LocalStorage support
* Tasks persist after page refresh

### 📱 Responsive Design

* Desktop friendly
* Tablet responsive
* Mobile responsive
